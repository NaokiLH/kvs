# Naive impl of talent-plan

[talent-plan](https://github.com/pingcap/talent-plan/blob/master/courses/rust/docs/lesson-plan.md#user-content-building-blocks-3)

### Install
```sh
cargo build --release
```
if you just want to test 
```sh
cargo test
```

### Getting Started
```sh
cd target/release
./kvs-client -h
```
